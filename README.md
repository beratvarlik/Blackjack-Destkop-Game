Blackjack Desktop Game (C#)

This project is a desktop Blackjack game developed using C# and Windows Forms.
The main purpose of this project is to practice object-oriented programming (OOP) concepts and desktop application development in .NET.

  Features
* Classic Blackjack game logic
* Single 52-card deck
* Random card shuffling
* Player vs Dealer gameplay
* Score calculation and win/lose detection
* Simple desktop user interface

Technologies Used
* C#
* .NET
* Windows Forms

   Project Structure
The project is designed with a simple object-oriented architecture:

* Card – Represents a playing card with suit, rank, and value.
* Deck – Creates and manages the 52-card deck and handles shuffling.
* Player – Stores the player's hand and calculates the score.
* Game Manager – Controls the game flow and dealer logic.

Purpose of the Project
This project was created to improve skills in:

* Object-Oriented Programming
* Game logic design
* Event-driven programming
* Desktop UI development

   Future Improvements

* Possible future features include:
* Multiple deck support
* Betting system
* Split and Double Down actions
* Card graphics instead of text
* Improved UI design

How to Run
* Clone the repository
* Open the solution in Visual Studio
* Build and run the project
